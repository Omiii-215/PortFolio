# The Portfolio | Developer Edition

Welcome to the Developer Edition of my portfolio—a meticulously designed hub mirroring the sharp, high-density aesthetics of print journalism. It documents my journey across open-source ecosystems, statistical machine learning pipelines, and robust full-stack systems.

## Features
- **Editorial Aesthetics:** Geometric layouts devoid of rounded corners, unified under a stark off-white/black color palette.
- **Responsive Newspaper Grid:** Built entirely utilizing raw CSS Grid architectures that gracefully collapse columns on mobile viewports.
- **Typographic Precision:** Highly structured typography hierarchies pairing classic Serif headlines with crisp Monospace metadata.
- **Mechanical Interactions:** Custom snappy intersection observers and hard offset-shadow hovers replacing generic floats and bounces.
- **Performant & Zero-Dependency:** Extremely lightweight architecture free of bloated CSS frameworks, maintaining high performance metrics.

## Tech Stack
- **HTML5:** Highly semantic and accessible markup structured natively.
- **CSS3:** Custom Newsprint design system variables, grids, flexboxes, and specialized texture layouts built from scratch without external frameworks.
- **JavaScript (ES6):** Vanilla logic for Intersection Observers and dynamic, non-intrusive interactive states.
- **Font Awesome:** Integrated for rigid, bold typography-matching brand icons.

## About the Portfolio
This site serves as my active digital ledger. It records my engagements with aggressive technical challenges like GSoC 2026 (Spectral Timing in Julia), Data Science exploration (Bike Analytics), and Web Cloud deployments (Bionium X). Built to be uncompromising in structure, it reflects an engineering philosophy prioritizing raw performance, clarity, and bold implementation.
