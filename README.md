# Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, experience, and projects as a Software Developer and Open Source Enthusiast. It serves as a central hub for my professional identity and a way to connect with others in the industry.

## üöÄ Features

-   **Responsive Design:** Fully responsive layout that adapts to all screen sizes, from mobile devices to large desktops.
-   **Dark/Light Mode:** Includes a theme toggle allowing users to switch between a sleek dark mode and a crisp light mode.
-   **Smooth Scrolling:** Enhanced navigation experience with smooth scrolling between sections.
-   **Scroll Animations:** Elements fade in and animate as you scroll down the page using the Intersection Observer API.
-   **Dynamic Content:** Auto-updating year in the footer.
-   **Interactive UI:** Hover effects on buttons, project cards, and skill items for a modern look and feel.

## üõ†Ô∏è Built With

This project is built using pure, vanilla web technologies to ensure high performance and full control over the design:

-   **HTML5:** Semantic markup for accessibility and structure.
-   **CSS3:** Custom styling using CSS Variables, Flexbox, and CSS Grid. No external CSS frameworks were used (except for Font Awesome icons).
-   **JavaScript (ES6+):** Vanilla JavaScript for DOM manipulation, event handling, and logic (theme toggle, mobile menu, observers).
-   **Font Awesome:** For icons used in the social links and UI elements.

## üîß Setup & Installation

Since this project uses static HTML, CSS, and JS, there is no need for a complex build process or package installation.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/portfolio.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd portfolio
    ```

3.  **Open the project:**
    -   Simply double-click `index.html` to open it in your default browser.
    -   Or, for a better development experience, use a live server (e.g., Live Server extension in VS Code).

## üé® Customization

-   **Colors:** You can easily change the color scheme by editing the CSS Variables defined in the `:root` and `[data-theme="light"]` blocks in `style.css`.
-   **Content:** Edit `index.html` to update the text, links, and content for your own portfolio.
-   **Fonts:** The project uses 'Outfit' and 'Roboto' from Google Fonts. You can change these in the `<head>` of `index.html`.
